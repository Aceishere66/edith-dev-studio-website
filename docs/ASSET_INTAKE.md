# Asset Intake

## Root Folder

Product assets belong under:

`public/assets/product/upscaler-studio/`

## Folders

- `screenshots/` - real application screenshots.
- `before-after/` - owned before/after comparisons.
- `mockups/` - product frames, desktop mockups, store graphics.
- `3d/` - future 3D or interactive assets if a later phase approves them.

## Required Assets

1. App screenshot
   - Suggested size: 2400 x 1500 or larger.
   - Format: PNG or WebP.
   - Naming: `upscaler-studio-app-main.png`.

2. Before/after comparison
   - Suggested size: 2000 x 1200 or larger.
   - Format: PNG or WebP.
   - Naming: `before-after-photo-restoration-01.png`.

3. Product mockup
   - Suggested size: 2400 x 1600.
   - Format: PNG or WebP.
   - Naming: `upscaler-studio-windows-mockup.png`.

4. Future 3D or interactive asset
   - Add only after performance and licensing review.
   - Naming should include source and version.

## Licensing Rules

- Use owned screenshots, internally generated product visuals, or commercially licensed source material.
- Do not use competitor screenshots, competitor imagery, random online photos, or unverified 3D assets.
- Record the source, license, and owner for every non-original asset in this document before publishing it.

## Current State

The site uses CSS-generated placeholders through `src/components/AssetPlaceholder.astro`. Replace those placeholders only when real product assets are ready.
