# EDITH Dev Studio website starter

Astro starter for a premium dark product website centered on **Upscaler Studio** and future EDITH Dev Studio products.

## Included pages

- `/`
- `/products/upscaler-studio`
- `/pricing`
- `/download`
- `/support`
- `/about`
- `/privacy`
- `/terms`
- `/cookies`

## Quick start

```bash
npm install
npm run dev
```
